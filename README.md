# ZEKPASS-1.0
ZEKPASS — Smart Event Ticketing Platform Une plateforme moderne de billetterie événementielle permettant aux organisateurs de créer, gérer et vendre des tickets en ligne avec QR Code sécurisé, gestion des participants, suivi des ventes et paiement digital. Conçu pour concerts, soirées, festivals, matchs, conférences et événements privés.
